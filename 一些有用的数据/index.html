一些有用的数据

May 6, 2018

出处

E5 2630

  • L2 cache access costs 4.3ns
  • L3 cache access costs 7.9ns
  • cache miss 32 ns
  • atomic Lock 8.23ns
  • spinlock/unock 16.1ns 34~39 cpu cycles (On the same cpu)
  • System call with SELINUX 75.34ns
  • System call without SELINUX 41.85ns

E5 2695

  • spin_{lock,unlock}: 13.94ns
  • local_BH_{disable,enable}: 18 cycles(tsc) 7.410 ns
  • local_IRQ_{disable,enable}: 7 cycles(tsc) 2.860 ns
  • local_IRQ_{save,restore}: 37 cycles(tsc) 14.837 ns
